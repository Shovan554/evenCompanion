# Relay Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Build the tiny cloud WebSocket fan-out service that carries Mac stat snapshots to the glasses and relays reverse commands back, deployable to Render.

**Architecture:** A transport-agnostic `Hub` holds, per secret-token "room", the latest snapshot and the set of connected clients (publishers and subscribers). A thin WebSocket `server` layer (Node `http` + `ws`) authenticates connections by `?token=&role=` query params and wires each socket into the Hub. The Hub fans publisher frames out to subscribers, forwards subscriber frames (commands) to publishers, drops malformed JSON, and evicts empty rooms. Render runs it as a free web service with a `/healthz` check.

**Tech Stack:** Node ≥20 (ESM), TypeScript, `ws`, `tsup` (build/bundle), `tsx` (dev), `vitest` (tests). Hosted on Render.

## Global Constraints

- Node `>=20`; package is ESM (`"type": "module"`).
- The relay is **pure transport** — it MUST NOT parse or depend on the snapshot/command schema beyond validating that a frame is syntactically valid JSON.
- A connection is identified by URL query params: `token` (secret, also the room key) and `role` (`pub` | `sub`); WebSocket path is `/ws`.
- When env `RELAY_TOKEN` is set, only that exact token is accepted; otherwise any token is accepted (each token is its own private room).
- Listen on `process.env.PORT` (Render injects it); default `8080` locally.
- Health check endpoint: `GET /healthz` → `200 "ok"`.
- All work lives under the `relay/` directory of the `macCompanion` repo.
- Frequent commits: one per task. Use `git -C` or run git from the repo root; the repo is at `macCompanion/` (branch `main`).

```ts
import { randomUUID } from 'node:crypto'

export type Role = 'pub' | 'sub'
export type SendFn = (data: string) => void

interface Client {
  id: string
  role: Role
  send: SendFn
}

interface Room {
  latest?: string
  clients: Map<string, Client>
}

export class Hub {
  private rooms = new Map<string, Room>()

  constructor(private idFactory: () => string = randomUUID) {}

  addClient(token: string, role: Role, send: SendFn): string {
    const id = this.idFactory()
    let room = this.rooms.get(token)
    if (!room) {
      room = { clients: new Map() }
      this.rooms.set(token, room)
    }
    room.clients.set(id, { id, role, send })
    if (role === 'sub' && room.latest !== undefined) {
      send(room.latest)
    }
    return id
  }

  handleMessage(clientId: string, raw: string): void {
    const found = this.findClient(clientId)
    if (!found) return
    const { room, client } = found
    if (client.role === 'pub') {
      room.latest = raw
      for (const c of room.clients.values()) {
        if (c.role === 'sub') c.send(raw)
      }
    }
  }

  latestSnapshot(token: string): string | undefined {
    return this.rooms.get(token)?.latest
  }

  roomCount(): number {
    return this.rooms.size
  }

  protected findClient(clientId: string): { room: Room; client: Client } | undefined {
    for (const room of this.rooms.values()) {
      const client = room.clients.get(clientId)
      if (client) return { room, client }
    }
    return undefined
  }

  protected get roomsMap(): Map<string, Room> {
    return this.rooms
  }
}
```

```ts
import { createServer, type IncomingMessage, type Server } from 'node:http'
import { WebSocketServer, type WebSocket } from 'ws'
import { Hub, type Role } from './hub'

export interface RelayServer {
  port: number
  close(): Promise<void>
}

export function createRelayServer(opts: { authToken?: string } = {}) {
  const hub = new Hub()

  const http: Server = createServer((req, res) => {
    if (req.method === 'GET' && req.url === '/healthz') {
      res.writeHead(200, { 'content-type': 'text/plain' })
      res.end('ok')
      return
    }
    res.writeHead(404)
    res.end()
  })

  const wss = new WebSocketServer({ server: http, path: '/ws' })

  wss.on('connection', (ws: WebSocket, req: IncomingMessage) => {
    const url = new URL(req.url ?? '', 'http://localhost')
    const token = url.searchParams.get('token') ?? ''
    const role = url.searchParams.get('role')

    const authorized =
      token.length > 0 &&
      (role === 'pub' || role === 'sub') &&
      (!opts.authToken || token === opts.authToken)

    if (!authorized) {
      ws.close(1008, 'unauthorized')
      return
    }

    const id = hub.addClient(token, role as Role, (data) => {
      if (ws.readyState === ws.OPEN) ws.send(data)
    })
    ws.on('message', (data) => hub.handleMessage(id, data.toString()))
    ws.on('close', () => hub.removeClient(id))
    ws.on('error', () => hub.removeClient(id))
  })

  return {
    listen(port: number): Promise<RelayServer> {
      return new Promise((resolve) => {
        http.listen(port, () => {
          const addr = http.address()
          const actualPort = typeof addr === 'object' && addr ? addr.port : port
          resolve({
            port: actualPort,
            close: () =>
              new Promise<void>((res) => {
                wss.close(() => http.close(() => res()))
              }),
          })
        })
      })
    },
  }
}
```
